Description of the tools

We are using Visual Studio Code as Integerated Developement Environment along with Intellij for development. The autocompelete and intellisense features help in writing javascript and html code in these IDE'S.

We are using Leaflet API for drawing maps and pin's on map for showing the crime data. It is really simple tool that let us plot and draw maps using simple javascript API. It is simple and easy to maintain

We are using Nominatim API to get the coordintes for locations from crime data so that we can plot them. Currently they have limitation's on their call If you make too many request it blocks you.



Six interesting findings that you discovered using your application

It was fun doing the project below are the interesting six things that I discovered while doing this project.

  • Auto-complete functionality of different IDE. It really help's and make your life easy
  • Leaflet and Nomination API makes things easy for working with maps. It's really simple API and you do not have to worry about details.
  • Node is quite extensive like other server side langauges. You can implement any kind of application in it.
  • Its still quite tough to debug error's in javascript.
  • Nomination API have limitations with number of requests you can make in a second. This can be annyoying.
  • Bootstrap helps with styling with the html of your application. You don't have to worry too much about style's
